## *Ferramentas...*
* node
* npm ou yarn
* git

## *Instalação*

Primeiro passo, clonar o projeto:
``` bash
# Clonar
git clone https://github.com/lecastro/site-pessoal

# Acessar
cd site-pessoal
```
## Configuração - Frontend
``` bash
# Atualizar dependências
yarn install

# Rodar em ambiente local localhost:8080
yarn run dev

# Rodar em ambiente de produção
yarn run build
